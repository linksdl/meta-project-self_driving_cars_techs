#### 3, how to make, install, debug a c++ code?
> 1, 在当前文件是C++的情况下，需要设置两个文件tasks.json，launch.json。
- tasks.json可以被用来做编译；
- launch.json用来执行编译好的文件

> 2,使用C/C++ Compile Run插件
- C/C++ Compile Run

> 3,使用cmake
- 单步调试一定要在CMakeLists.txt中设置 set(CMAKE_BUILD_TYPE DEBUG)
